### GIWIFI
----
**GIWIFI.py** 
* 模拟正常登录
* 文件中输入账号密码运行即登录
----
**破解方式**  
> 由于很多学校的封禁程度不同，需要自行测试
* 53端口，需要 服务器 + OpenVPN + VpnsMgr
* http://down.gwifi.com.cn/getApp.htm?os=ios 每次请求这个链接会获得十五分钟的网络使用，可做一个脚本不断请求链接
* wifidog有一个阻塞引起的BUG
> 第一种和第二种方式我都用了一个月左右就被他们封禁了，第三种方式就暂时不公布脚本了(考研狗近一年需要Wifi用，流量卡的一匹)
